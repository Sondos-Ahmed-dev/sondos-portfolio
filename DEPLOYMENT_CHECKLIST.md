# قائمة التحقق قبل النشر

## ✅ قبل البدء في النشر

### الاختبارات المحلية
- [ ] تشغيل `pnpm run build` بنجاح
- [ ] عدم وجود أخطاء TypeScript (`pnpm run check`)
- [ ] جميع الاختبارات تمر (`pnpm test`)
- [ ] اختبار الموقع محلياً (`pnpm run preview`)
- [ ] اختبار نموذج التواصل
- [ ] اختبار تحميل السيرة الذاتية
- [ ] اختبار جميع الروابط الخارجية

### التحقق من المحتوى
- [ ] جميع الروابط صحيحة (GitHub, LinkedIn, Email)
- [ ] لا توجد أخطاء إملائية
- [ ] جميع الصور محسّنة
- [ ] جميع الأيقونات تحميل بشكل صحيح
- [ ] الفافيكون يظهر بشكل صحيح

### الأداء والأمان
- [ ] تشغيل Lighthouse audit
- [ ] FCP < 1.5s
- [ ] LCP < 2.5s
- [ ] CLS < 0.1
- [ ] TTI < 3.5s
- [ ] جميع رؤوس الأمان موجودة

### SEO والوصول
- [ ] Meta tags صحيحة
- [ ] JSON-LD Structured Data موجود
- [ ] robots.txt موجود
- [ ] sitemap.xml موجود
- [ ] WCAG 2.1 AA compliance
- [ ] Keyboard navigation يعمل
- [ ] Screen reader compatible
